# 快速开始指南

欢迎开始Java架构学习之旅！本指南将帮助你快速上手这个学习系统。

## 🎯 第一步：了解你的学习系统

你的学习仓库包含7个核心目录：

| 目录 | 用途 | 更新频率 |
|------|------|----------|
| **01-plans** | 存放所有学习计划 | 每周/每月 |
| **02-progress** | 追踪学习进度 | 每天 |
| **03-notes** | 记录学习笔记 | 每学一个知识点 |
| **04-examples** | 存放示例代码 | 每学一个知识点 |
| **05-projects** | 完成的项目 | 每个项目 |
| **06-resources** | 学习资源整理 | 随时 |
| **07-review** | 复习与总结 | 每周/每月 |

## 📋 第二步：查看总体学习计划

## 📚 第五步：开始学习

### 学习流程（每个知识点）

#### 1. 理论学习（1小时）
- 看视频/读文档学习概念
- 查看 `03-notes/` 中对应的知识点笔记
- 例如：`03-notes/01-java-basics/collections.md`

#### 2. 代码练习（2小时）
- 在 `04-examples/` 中编写示例代码
- 每个知识点至少写3个示例
- 确保代码能运行

#### 3. 笔记记录（30分钟）
- 更新 `03-notes/` 中的笔记
- 使用模板：概念、要点、AI对话、代码、心得

#### 4. 与AI互动（30分钟）
- 问AI不懂的问题
- 让AI出练习题
- 让AI Review你的代码
- 记录有价值的对话到 `06-resources/ai-chats/`

#### 5. 项目实践（根据进度）
- 在 `05-projects/practice/` 完成练习项目
- 在 `05-projects/integrated/` 完成综合项目

## 💡 第六步：有效使用AI

### 查看AI使用指南

```bash

```

### AI提问模板

**学习新概念**：
```
我是Java初学者，请用通俗语言解释ArrayList，
最好用生活中的例子，并给一个简单的代码示例。
```

**代码审查**：
```
请帮我Review这段代码：
[粘贴代码]

从以下角度分析：
1. 代码规范
2. 潜在bug
3. 性能问题
4. 改进建议
```

**生成练习题**：
```
我刚学完HashMap，给我5道练习题，
从易到难，每题都要有实际应用场景。
```

## 🔄 第七步：定期复习

### 每日复习（每天晚上，10分钟）
- 回顾今天学的知识点
- 看一眼 `03-notes/` 中的笔记
- 在脑海中复述一遍

### 每周总结（每周日，30分钟）
- 在 `07-review/weekly/` 创建本周总结
- 统计学习时长、代码量、完成的知识点
- 总结遇到的问题和解决方案
- 制定下周计划

### 每月总结（每月底，1小时）
- 在 `07-review/monthly/` 创建本月总结
- 评估月度目标完成情况
- 调整下月计划
- 更新技能树

### 知识点复习（使用间隔重复法）
- 第1次复习：学习后1天
- 第2次复习：学习后3天
- 第3次复习：学习后1周
- 第4次复习：学习后1个月
- 第5次复习：学习后3个月

在 `03-notes/` 每个知识点的笔记中记录复习计划。

## 🆘 遇到问题怎么办？

### 1. 先问AI
- 具体描述问题
- 提供错误信息和代码
- 说明你的环境

### 2. 再查文档
- Java官方文档
- Spring官方文档
- 框架官方文档

### 3. 最后搜索
- Google/百度搜索
- Stack Overflow
- GitHub Issues

### 4. 记录解决方案
- 在 `06-resources/ai-chats/` 中记录
- 标注问题、原因、解决方案
- 下次遇到类似问题就能快速解决
