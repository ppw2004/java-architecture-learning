# 学习计划

## 目录说明

本目录用于存放学习计划，包括总体规划和每日学习安排。

### 文件结构

```
01-plans/
├── README.md                 # 本文件
├── master-plan.md           # 总体学习计划（3-12个月）
├── phase1-plan.md           # 第一阶段计划（0-3个月）
├── phase2-plan.md           # 第二阶段计划（4-6个月）
├── phase3-plan.md           # 第三阶段计划（7-12个月）
└── daily/                   # 每日学习计划
    ├── 2026-01/
    ├── 2026-02/
    └── ...
```

## 使用说明

1. **总体规划**：制定3-12个月的长期学习目标
2. **阶段计划**：将长期目标分解为3个阶段
3. **月度计划**：每月初制定当月学习重点
4. **周计划**：每周日制定下周学习任务
5. **日计划**：每天早上制定当天学习任务

## 计划制定原则

- **SMART原则**：具体、可衡量、可达成、相关性、时限性
- **二八定律**：80%精力放在20%核心知识点上
- **循序渐进**：先易后难，打好基础
- **理论实践结合**：每个知识点都要有代码实践
- **定期回顾**：每周总结，每月调整计划
