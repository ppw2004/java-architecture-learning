# 学习进度追踪

## 目录说明

本目录用于追踪学习进度，包括已完成的知识点、项目进度、技能掌握情况等。

### 文件结构

```
02-progress/
├── README.md                # 本文件
├── checklist.md             # 知识点清单（打勾式）
├── timeline.md              # 学习时间线
├── milestones.md            # 重要里程碑
├── skills.md                # 技能掌握度评估
└── projects.md              # 项目完成情况
```

## 进度追踪方式

### 1. 知识点清单
- ✅ 已完成
- 🟡 进行中
- ⏸️ 暂停
- ❌ 未开始

### 2. 掌握度评估
- ⭐⭐⭐⭐⭐ 精通（能教别人）
- ⭐⭐⭐⭐ 熟练（能独立使用）
- ⭐⭐⭐ 掌握（理解并能简单使用）
- ⭐⭐ 了解（知道概念）
- ⭐ 未接触

### 3. 统计指标
- 已学习知识点数 / 总知识点数
- 代码行数统计
- 项目完成数
- 学习天数

## 定期更新

- **每日更新**：更新当天完成的任务
- **每周总结**：每周日总结本周进度
- **月度评估**：每月底评估月度目标完成情况
