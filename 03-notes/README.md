## 笔记模板

每个知识点的笔记应包含以下部分：

```markdown
# [知识点名称]

## 📚 概念理解
[用自己的话解释概念]

## 🎯 核心要点
- 要点1
- 要点2
- 要点3

## 💡 AI指导
[与AI讨论的重要问题和答案]

## 💻 示例代码
```java
[代码示例]
```

## 📝 学习心得
[自己的理解和体会]

## ❓ 疑问与解答
[学习过程中的疑问和解答]

## 🔗 相关资源
- [ ] 官方文档
- [ ] 视频教程
- [ ] 博客文章
- [ ] 开源项目

## ✅ 练习题
- [ ] 练习1
- [ ] 练习2
- [ ] 练习3
```

## 使用建议

1. **及时记录**：学完一个知识点立即记录
2. **用自己的话**：不要简单复制，要理解后重新组织
3. **代码为王**：每个概念都要有代码示例
4. **AI互动**：记录与AI的有价值对话
5. **定期复习**：使用间隔重复法复习笔记
